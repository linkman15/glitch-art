For this project I used Audacity to databend the Audacity Logo into multiple different pictures using different Audacity effects

I started by first converting the Audacity Logo from a .bmp file to raw data in Audacity.

I then highlighted a section of the resulting sound file and added various effects to it, such as echo or bass.

I would save the result each time to a .bmp resulting in the glitched image.

I repeated this process multiple times, adding more effects. -James Cole
